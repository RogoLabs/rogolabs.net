About Jerry Gamblin

Creating practical vulnerability intelligence solutions for security professionals

Building Open Source Security Intelligence

I've spent my career obsessed with one question: How do we fix what matters? This drove me to create RogoLabs—named after the Latin word "rogo," meaning "to perceive"—where I develop free, open source tools that bring clarity to the chaos of vulnerability management.

While others get lost in endless CVE feeds and alert fatigue, I build practical tools that cut through the noise. My open source projects transform overwhelming vulnerability data into clear, actionable intelligence that security teams can actually use. No vendor lock-in, no hidden costs—just effective tools shared freely with the community.

I believe the best security happens when we collaborate openly. Every vulnerability intelligence tool I create is released as open source because when security knowledge is shared freely, we all become stronger. The community's feedback and contributions make these tools better for everyone.

The mission is simple: Create open source vulnerability intelligence tools that help security professionals perceive what matters, prioritize what's critical, and protect what counts.
